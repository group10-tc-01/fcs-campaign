# FCS Campaign

O FCS Campaign gerencia campanhas, disponibiliza transparência pública e valida a elegibilidade de uma campanha para doações.

## Operação

- Namespace Kubernetes: `fcs-campaigns`
- Health check: `/health`
- Métricas: `/metrics`
- API: `/api/v1`

Consulte o [README](../README.md) para requisitos, execução local e contratos da API.
